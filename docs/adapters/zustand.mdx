## Installation

```bash
npm install @estedio/react axios zustand
```

## Setup

The Zustand adapter creates one vanilla Zustand store per resource and exposes typed dispatch/selector hooks.

### 1. Create stores and hooks

```ts
"use client";

import {
  createStore,
  createHooks,
  localStorageAdapter,
} from "@estedio/react/zustand";
import { resources } from "./resources";
import type { ResourceTypeMap } from "./resources";

const { stores } = createStore<ResourceTypeMap>(resources, {
  storageByResource: {
    GetUsers: localStorageAdapter,
  },
});

export const { useResourceDispatch, useResourceSelector } =
  createHooks<ResourceTypeMap>(stores);
```

`createResources` and `createStore` are equivalent. `createResourceHooks` and `createHooks` are equivalent.

### 2. Optional provider

`ZustandProvider` is optional. It does not create context state, but it does let you initialize shared library config in the same pattern as the Redux and TanStack adapters.

```tsx
"use client";

import type { ReactNode } from "react";
import { ZustandProvider } from "@estedio/react/zustand";

export function AppProvider({ children }: { children: ReactNode }) {
  return (
    <ZustandProvider provider={{ apiBaseUrl: "https://api.example.com" }}>
      {children}
    </ZustandProvider>
  );
}
```

### 3. Use in components

```tsx
"use client";

import { useResourceDispatch, useResourceSelector } from "./store";

export function UserList() {
  const dispatch = useResourceDispatch();
  const { result, loading, error } = useResourceSelector("GetUsers");

  return (
    <>
      <button onClick={() => dispatch({ resource: "GetUsers", force: false })}>
        Load users
      </button>
      {loading && <p>Loading...</p>}
      {error && <p>Error: {error}</p>}
      <ul>
        {result?.map((user) => (
          <li key={user.id}>{user.name}</li>
        ))}
      </ul>
    </>
  );
}
```

## Dispatching payloads and callbacks

```ts
dispatch({ resource: "GetUserById" }, { userId: 42 });

dispatch(
  { resource: "CreateUser" },
  { name: "Alice", email: "alice@example.com" },
  (result, error) => {
    console.log(result, error);
  },
);
```

## Cache and persistence behavior

- `force: true` always fetches
- `force: false` skips the request when a non-null result already exists
- `meta.persist` and `meta.persistKey` enable encrypted persisted hydration
- `meta.isAuthResource` saves `access_token` and `refresh_token` responses through the shared token manager

```ts
const { stores } = createStore<ResourceTypeMap>(resources, {
  storageByResource: {
    GetUsers: localStorageAdapter,
    GetProfile: sessionStorageAdapter,
  },
});
```

## Advanced types

```ts
import type { ResourceStoresFromTypeMap } from "@estedio/react/zustand";
import type { ResourceTypeMap } from "./resources";

type AppStores = ResourceStoresFromTypeMap<ResourceTypeMap>;
```
